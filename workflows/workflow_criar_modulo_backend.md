# Workflow — Criar módulo backend

## Entrada necessária

- Nome do módulo.
- Campos da tabela.
- Endpoints esperados.
- Regras de autorização.
- Critérios de aceite.

## Passos

1. Ler regras de backend e banco.
2. Criar migration.
3. Criar queries sqlc.
4. Gerar código sqlc.
5. Criar repository se houver camada adicional.
6. Criar service.
7. Criar DTOs e validators.
8. Criar handler REST.
9. Atualizar rotas.
10. Criar testes focalizados.
11. Atualizar OpenAPI.
12. Rodar testes.

## Saída esperada

```text
Resumo
Arquivos alterados
Testes executados
Como validar
Pendências
```
