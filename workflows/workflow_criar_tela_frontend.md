# Workflow — Criar tela frontend

## Entrada necessária

- Contrato da API.
- Permissões necessárias.
- Campos do formulário.
- Filtros de listagem.
- Estados esperados.

## Passos

1. Ler tipos/contrato da API.
2. Criar `types.ts`.
3. Criar `schemas.ts` com Zod.
4. Criar `api.ts`.
5. Criar hooks com TanStack Query.
6. Criar componentes.
7. Criar página.
8. Adicionar rota.
9. Validar loading, erro, vazio e sucesso.

## Saída esperada

- Tela funcional.
- Validação de formulário.
- Cache e invalidação.
- Sem regra sensível duplicada no frontend.
